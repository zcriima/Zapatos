CAMBIAR PRODUCTOS
Cada producto tiene esta estructura:

{
  id: 1,
  name: "Zapato Mujer 1",
  category: "Mujer",
  sizes: "35 - 36 - 37",
  price: "$40",
  image: "images/mujer1.jpg"
}

Puedes cambiar:
- name
- category
- sizes
- price
- image
